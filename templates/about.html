Project Overview

The Disease Prediction System is an AI-powered web application designed to help users assess potential health risks based on symptoms. The system leverages machine learning algorithms trained on large medical datasets to provide accurate predictions quickly.

Key Features

  • Symptom-based Prediction: Users can select symptoms from a predefined list or manually input them for personalized disease predictions.
  • Real-time Analysis: Predictions are processed instantly without storing personal data, ensuring both speed and privacy.
  • High Accuracy: The system utilizes advanced ML models to achieve high prediction reliability.
  • User-friendly Interface: Designed to be intuitive and responsive across devices, ensuring a seamless experience on desktops, tablets, and mobiles.
  • Open Source: The source code is freely available, promoting transparency and community contributions.

Technology Stack

  • Frontend: HTML, CSS, JavaScript, Font Awesome
  • Backend: Flask (Python)
  • Machine Learning: Python (Scikit-learn, Pandas, NumPy)
  • Data Handling: JSON for symptom input/output
  • Deployment: Local server or cloud hosting

How It Works

Users select or enter symptoms into the web interface. The system then sends this information to the backend, where a trained ML model evaluates the data and returns the most probable disease(s). Results are displayed in an easy-to-understand format, highlighting the predicted disease and additional information for guidance.
